================================================================================
MENU MANAGEMENT - BACKEND VALIDATION FIX
================================================================================

ISSUE:
PUT https://stage-account.besewonline.com/api/v1/menu-config/{menuId} 400 (Bad Request)
Error: "Input validation failed"

ROOT CAUSE:
The backend API has strict input validation that requires ALL fields to be
present in the PUT request, not just the fields being updated. Sending only
partial fields causes validation to fail.

SOLUTION APPLIED:
Modified the update payload to send ALL menu fields, not just the ones being
edited. This ensures the backend validation passes.
