================================================================================
WHAT YOU CAN DO
================================================================================

SEED DEFAULT MENUS:
- Click "Seed Default Menus" button (top right)
- Confirm in dialog
- Default menus will populate

TOGGLE MENU STATUS:
- Find a menu in the table
- Click the toggle switch in "Active" column
- Menu status updates instantly

EDIT MENU:
- Click "Edit" button for any menu
- Modify: Label, Path, Min Trust Score
- Select: User Types, Worker Types, Subscription Tiers
- Click "Save Changes"
- Changes persist after page refresh

DELETE MENU:
- Click "Delete" button for any menu
- Confirm in dialog
- Menu is removed

VIEW HIERARCHY:
- Parent menus display first
- Child menus are indented with "└─" prefix
- Automatic sorting by hierarchy level
